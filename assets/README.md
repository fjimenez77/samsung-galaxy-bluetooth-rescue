# Brand & assets

Logo and banner for **Samsung Galaxy — Bluetooth Diagnostic & Rescue**.

- `logo.svg` — square app-style icon: gradient tile, Bluetooth glyph, a foldable hinge crease,
  and a green diagnostic "pulse."
- `banner.svg` — README header (title, tagline, feature chips).

## Palette
| Role | Color |
|---|---|
| Primary blue (gradient top → bottom) | `#2BA6FF` → `#0B53D0` |
| Diagnostic pulse accent | `#5BF0CE` |
| Banner background | `#0A1426` → `#0E1B33` |
| Chip fill / stroke | `#13294F` / `#27508F` |

Both files are hand-authored, raster-free **SVG**, so they scale crisply at any size — edit the
SVG source directly rather than exporting from a design tool.
